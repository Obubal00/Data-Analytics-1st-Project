# 📊 Customer Data Cleaning & Analysis Project

## 🎯 Project Overview
This project demonstrates professional data cleaning and analysis skills using Google Sheets/Excel. I transformed a messy customer dataset into a clean, analyzable format and generated actionable business insights.

## 📁 Dataset Description
- **Source:** Customer database with 100+ records
- **Original Issues:** 
  - Inconsistent formatting
  - Missing values
  - Mixed date formats
  - Unstructured text data

## 🛠️ Tools Used
- Google Sheets / Microsoft Excel
- Pivot Tables
- Advanced Formulas (VLOOKUP, IF, TEXT, PROPER)
- Data Visualization

## 📋 Data Cleaning Steps Performed
1. **Standardized Name Formatting:** Used PROPER() to fix casing
2. **Validated Email Addresses:** Checked for proper @domain.com format
3. **Formatted ZIP Codes:** Standardized to 5-digit format
4. **Fixed Date Formats:** Converted mixed date formats to standard
5. **Handled Missing Values:** Filled blanks with appropriate values
6. **Standardized Status Values:** Cleaned and normalized status categories
7. **Created Customer Tiers:** Segmented customers by spending
